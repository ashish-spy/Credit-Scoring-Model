# Credit Scoring Prediction using Machine Learning

## Overview

This project predicts whether a customer has Good Credit Risk or Bad Credit Risk using Machine Learning classification algorithms.

The project includes a Flask web application where users can enter customer information and receive an instant prediction.

- Feature Scaling
- Logistic Regression
